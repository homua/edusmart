# **App Name**: EduSmart Cloud

## Core Features:

- Real-time Data Sync: Synchronize users, classes, assignments, and submissions data in real-time using Firestore.
- User Authentication: Allow users to log in with their username and password. Support different user roles: Admin, Teacher, and Student.
- Admin Dashboard: Enable admins to manage users, classes, and import/export data to JSON files.
- Teacher Dashboard: Provide teachers with a dashboard to manage assignments, view reports, and manage their class roster.
- Student Portal: Offer students a portal to view assignments and submit their solutions.
- AI Student Parser: Provide a tool to parse a list of student names from bulk input using generative AI. LLM uses reasoning to parse.
- Data Backup/Restore: Provide functionality to backup and restore all application data using JSON file.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a smart and modern feel.
- Background color: Very light gray (#F9FAFB), almost white, providing a clean canvas for the content.
- Accent color: Emerald green (#34D399) to highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, geometric icons from a consistent set for navigation and actions.
- Subtle animations on transitions and interactions to enhance user experience.
